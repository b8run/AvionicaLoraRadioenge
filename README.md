# AvionicaLoraEbyte
Este repositório contém o código-fonte de um sistema aviônico focado na coleta de dados e controle de funções de forma remota. O sistema estabelece comunicação entre pontos master e slave utilizando o protocolo LoRaMesh com módulo RadioEnge.

Estrutura e Execução:

Código Principal: O projeto é composto por dois arquivos .ino.

Ambiente de Desenvolvimento: Os arquivos devem ser compilados na Arduino IDE para a placa ESP32.

Dependências: Todas as bibliotecas necessárias para o funcionamento do projeto estão localizadas na pasta libraries.
